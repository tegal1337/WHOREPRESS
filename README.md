> Wordpress login checker

## Install

```sh
npm install --global whorepress
```

## Usage

make sure you have a list with format

`url:username:pass
`

```sh
whorepress list.txt 
```
## Arguments list

| Arguments | Description | 
|----------|----------|
| --admin-only   | check only for admin priviledge   | 
| --debug    | debug error output   | 
| --output    | Output file   |
